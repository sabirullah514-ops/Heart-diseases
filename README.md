# ❤️ Heart Disease Prediction

## 🩺 Machine Learning Project

**👨‍💻 Developed by:** **Sabir Shah**

---

## 📖 About the Project

This project uses the **K-Nearest Neighbors (KNN)** machine learning algorithm to predict whether a person is at risk of heart disease based on their medical information. The model is deployed using **Streamlit**, providing a simple and interactive web interface for real-time predictions.

---

## ✨ Features

* ❤️ Heart disease prediction
* 🤖 K-Nearest Neighbors (KNN) model
* 🌐 Interactive Streamlit web application
* ⚡ Real-time predictions
* 📊 User-friendly interface

---

## 📁 Project Files

| File                  | Description                      |
| --------------------- | -------------------------------- |
| `heart_disease_pp.py` | Streamlit web application        |
| `Heart_Disease.ipynb` | Data analysis and model training |
| `knn_heart_model.pkl` | Trained KNN model                |
| `heart_scaler.pkl`    | Feature scaler                   |
| `columns.pkl`         | Feature column names             |

---

## 🚀 How to Run

```bash
git clone https://github.com/sabirullah514-ops/Heart-diseases.git
cd Heart-diseases
pip install -r requirements.txt
streamlit run heart_disease_pp.py
```
